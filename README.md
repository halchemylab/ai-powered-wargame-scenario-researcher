# AI-Powered Wargame Scenario Researcher

A modular Streamlit application that uses OpenAI's GPT-4o to generate and visualize tactical wargame scenarios on a 20x20 grid.

## Features

- **AI-Generated Scenarios:** Generates tactical terrain and unit maneuvers based on natural language research topics.
- **Interactive Map:** Visualization using Plotly with tactical terrain (Open, Water, Urban, Forest) and unit overlays.
- **Step-by-Step Playback:** Navigate through tactical frames generated by the AI.
- **Commander's Journal:** Export the full scenario briefing as a Markdown file.
- **Session Persistence:** Tracks metrics and maintains state across interaction steps.

## Installation

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Set up your OpenAI API Key:
    *   Create a `.env` file in the root directory with `OPENAI_API_KEY=your_key_here`, OR
    *   Enter it directly in the application sidebar.

## Usage

Run the Streamlit app:

```bash
streamlit run app.py
```

## Project Structure

*   `app.py`: Main application entry point.
*   `engine/`: Core logic modules.
    *   `ai_handler.py`: OpenAI integration using Structured Outputs.
    *   `map_renderer.py`: Plotly visualization logic.
*   `utils/`: Helper modules.
    *   `state_manager.py`: Streamlit session state management.
    *   `exporter.py`: Report generation logic.
